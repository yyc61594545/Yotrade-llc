## 🤖 Cursor：从"请求次数"改成了"用量额度"

Cursor 是目前最火的 AI 原生 IDE。2025 年中它把计费从"固定请求次数"换成了**额度（credit）制**——你的订阅费等额折算成用量额度，跑模型时按 API 单价扣。

| 套餐 | 月费 | 包含 |
|---|---|---|
| **Hobby（免费）** | $0 | 有限的 Tab 补全 + Auto 模式 |
| **Pro** | $20 | $20/月 用量额度 + 无限 Tab + 无限 Auto |
| **Pro+** | $60 | 3× 额度（$60/月）|
| **Ultra** | $200 | 20× 用量、新功能优先 |

💡 **怎么理解这 $20 额度**：按官方/社区口径，$20 大约够跑 **225 次 Claude Sonnet 4.5** 或 **约 500 次 GPT-5** 请求。Tab 补全和 Auto 模式几乎不耗额度，真正烧钱的是你手动点 Sonnet / Opus 这类旗舰模型跑 agent。

⚠️ **隐藏坑**：额度制意味着"重度用 agent 跑前沿模型"的人，$20 可能撑不到月底。如果你天天让它自动重构整个项目，要么升 Pro+，要么直接上 Ultra。轻度用户（主要靠 Tab 补全）则 Pro 绰绰有余。

## 🐙 GitHub Copilot：最便宜的入门，但 6 月要变天

Copilot 胜在**便宜 + 深度集成 VS Code / JetBrains**。它目前用"premium 请求"配额，但**官方已公告 2026 年 6 月 1 日起转向用量计费（AI Credits）**，按 token 实际消耗算。

| 套餐 | 月费 | premium 请求 | 模型 |
|---|---|---|---|
| **Free** | $0 | 50 次 agent/chat + 2,000 次补全 | Haiku 4.5、GPT-5 mini 等 |
| **Pro** | $10/座 | 300/月 | Anthropic / Google / OpenAI |
| **Pro+** | $39/座 | 1,500/月（5×）| 全模型，含 **Claude Opus 4.7** + GitHub Spark |
| **Business** | $19/座 | — | 团队管理 |
| **Enterprise** | $39/座 | — | 企业级 |

⚠️ **2026-06-01 计费变更**：GitHub 官方博客确认，所有 Copilot 套餐将从"数请求次数"切换到"按 token 用量"。月付 Pro / Pro+ 用户 6 月 1 日自动迁移；年付用户到期前维持原 premium 请求模式。**价格不变，但同样的钱能跑多少，要重新算账**——社区对此争议不小，迁移前建议关注自己的实际用量。

✅ **适合谁**：预算敏感、主要要"补全 + 偶尔问问题"的人，$10 的 Pro 是全场最低门槛。想用 Opus 4.7 跑重活，得上 $39 的 Pro+。

## 🧠 Claude Code：最强 agent，但国内付不了款

Claude Code 是 Anthropic 的终端/CLI 编程 agent，跑的是原生 Claude 模型，公认在**大型代码库自动重构、多文件 agent 任务**上最强。它不是单独买，而是**包含在 Claude 订阅里**。

| 套餐 | 月费 | 用量 |
|---|---|---|
| **Pro** | $20 | 入门额度，**5 小时滚动窗口** + 周限额 |
| **Max 5x** | $100 | ≈ 5× Pro 用量 |
| **Max 20x** | $200 | ≈ 20× Pro 用量 |

💡 **限额机制**：Anthropic 用"每 5 小时滚动窗口"重置额度，外加**每周总限额**（Max 还有单独的 Sonnet 周限额）。社区实测 Max 5x 约 88,000 token/5 小时、Max 20x 约 220,000 token/5 小时（非官方数字，仅供参考）。2026 年起所有付费档加了**"超额开关"**：用超后可选择按标准 API 单价继续，且能自己设月度封顶。

❌ **致命点（国内用户必看）**：**Anthropic 至今不开放中国大陆和香港的 Claude 订阅**——连港卡都搞不定。要用 Claude Pro / Max（从而用 Claude Code），**必须有美国身份链路（美卡 + 美国地址 + 美国手机号），或者走代订**。这一点和我们之前在 [《没有美卡也能订阅 ChatGPT / Claude / Cursor》](/blog/ai-subscription-payment-paths) 里讲的完全一致，是四款工具里支付门槛最高的。

## 🌊 Windsurf：2026 年三月刚改版

Windsurf（原 Codeium 的 IDE）主打 Cascade agent，常被拿来和 Cursor 对比。它在 **2026 年 3 月 19 日做了一次结构性改版**，把原来的 credit 制调整成了配额制，部分套餐价格也动过。

| 套餐 | 月费 | 包含 |
|---|---|---|
| **Free** | $0 | 有限 Cascade 会话 + 无限 Tab |
| **Pro** | ~$15 | 无限 Cascade + 全部模型 |
| **Pro Plus** | ~$35 | 旗舰模型优先访问 |
| **Teams** | ~$25/座 | 管理后台 + SSO |

⚠️ **数据时效提醒**：Windsurf 这次改版后，不同来源对 Pro 到底是 $15 还是 $20、是 credit 还是配额说法不一。**订阅前务必去 windsurf.com/pricing 核对当下数字**，本文只给区间。它最大的优势是**免费档和低价 Pro 都能玩 agent IDE**，适合预算有限又想体验 Cursor 同类产品的人。

## 进阶：能不能组合用 + 怎么省钱

很多重度用户其实**不是二选一，而是组合**。常见搭配：

- 💡 **Copilot Pro（$10）打底 + Claude Code（按需）攻坚**：日常补全和小问题用便宜的 Copilot，遇到大重构/复杂 agent 任务才调 Claude Code。两者形态不同（插件 vs 终端），不冲突。
- 💡 **Cursor Pro 主力 + Windsurf Free 备用**：两款都是独立 IDE，用 Cursor 跑主力、Windsurf 免费档当第二意见。
- 💡 **团队场景**：Copilot Business（$19/座）和 Cursor Teams 更适合统一计费 + 后台管理，比每人各买个人版好对账。

省钱的几个实操点：

| 技巧 | 说明 |
|---|---|
| **先用免费档摸用量** | Copilot Free、Windsurf Free 都能先跑一阵，估出自己每月真实消耗再决定升不升 |
| **额度制别乱点旗舰模型** | Cursor 的额度主要被手动选 Opus/Sonnet 跑 agent 烧掉，日常 Tab + Auto 几乎不耗 |
| **盯紧 Copilot 6 月变更** | 转用量计费后"同价能跑多少"会变，迁移前后各记一周用量对比 |
| **按需而非长期囤** | 短期项目用代订按月付、随时停，比为一个月需求建一整套美国链路省得多 |

⚠️ **一个反直觉的点**：最贵的不一定最合适。如果你 80% 时间只是要代码补全，$200 的 Claude Code Max 20x 完全是浪费——那 $10 的 Copilot Pro 才是对的答案。**先量用量，再选档**。

## 国内用户怎么订阅这些工具？

这四款工具的官方支付几乎都要**境外信用卡 + 境外账单地址**，Claude 更是连港卡都不认。可行的合规路径有三条（详细成本拆解见 [《4 种合规支付路径对比》](/blog/ai-subscription-payment-paths)）：

| 路径 | 适合人群 | 说明 |
|---|---|---|
| **美卡 + 美国地址链路** | 长期重度用户 | 一劳永逸，**唯一能直付 Claude** 的方案 |
| **港卡 + 港区切换** | 中度用户 | 能订 Cursor/Copilot/Windsurf，**但 Claude 不行** |
| **第三方代订（YoTrade）** | 短期 / 小白 / 想用 Claude | 一对一专属账号，省去办卡 |

✅ **推荐做法**：

- 如果你已经是跨境开发者、迟早要 [办美卡建美国身份链路](/blog/nra-us-credit-card-12-month-roadmap)，那就一步到位——美卡能直付全部四款，还顺手薅开卡奖励。
- 如果只是想用 Cursor / Copilot / Windsurf，手上有 [港卡](/blog/hk-bank-card-guide) 也能覆盖大部分。
- 如果你**就是想用 Claude Code 这款最强 agent**，又不想为它单独建一整套美国身份——代订是当下最省事的解。

⚠️ **别碰的坑**：低于市场价一半的"号池共享"代订，本质是一张美卡刷上百个账号，Anthropic/OpenAI 一旦批量风控，你的账号和聊天记录一起没。认准**自有美卡链路、一对一专属**的服务商。

## ❓ 常见问题

**Q：四款里哪款最适合完全新手？**
A：GitHub Copilot。$10 门槛最低，直接装在 VS Code 里就能用，补全体验顺滑，犯错成本低。等用熟了再考虑要不要上 agent 更强的 Claude Code 或 Cursor。

**Q：Claude Code 真的国内一点办法都没有吗？**
A：官方渠道确实不开放中国大陆和香港订阅，**港卡也不行**。要用只有两条路：自建美国身份链路（美卡 + 美国地址 + 美国手机号），或者找一对一专属的代订。这也是它和另外三款最大的区别。

**Q：年付能省多少？**
A：各家年付通常比月付有折扣，但 AI 工具计费方式 2026 年变动太频繁（Copilot 6 月就要转用量计费），**长期锁定年付有踩"改版后更不划算"的风险**。建议先月付跑稳一两个月，确认长期需求再考虑年付。

**Q：代订安全吗？会不会被封号？**
A：风险全在服务商质量。用**号池共享卡**的低价代订最容易被批量风控；认准**自有美卡链路 + 一对一专属账号**（你自己的邮箱密码）的服务商，账号归属清晰，售后也有保障。
