Cursor 自 2025 年 6 月调整计费模型以来，定价架构稳定下来了：**所有付费档位都包含一个等额的"月度 credit pool"**，用于调用付费模型（Claude Sonnet 4.6、GPT-5.4、Gemini 3.1 等）。Tab 补全和 Auto mode 本身是无限的，credit pool 只对调用"premium model"扣费。先看官方价格。

## 中国大陆 IP 限制：怎么避坑

2025 年 7 月起，Cursor 在反爬虫和滥用治理收紧之后，**对来自中国大陆 IP 的账户，会限制访问 Claude / GPT-5.4 / Gemini 3.1 等核心模型**，只允许使用 Cursor 自研的 Tab 模型。

⚠️ **触发条件**（YoTrade 实战观察）：

- ❌ 用大陆 IP 注册账户
- ❌ 注册时用的卡是国内虚拟卡 / 银联通道
- ❌ 同一账户 30 天内频繁切换"大陆 ↔ 海外"IP

✅ **避坑组合（强烈推荐）**：

1. **注册 IP**：稳定的香港 / 日本 / 新加坡 / 美国节点
2. **付款卡**：美卡（最优）/ 港卡（次优）/ WildCard 虚拟卡（应急）
3. **日常使用 IP**：与注册 IP 一致，至少同地区
4. **不要**同一个账户家里 / 公司 / 咖啡店 IP 来回切

📌 **已经被降级了怎么办**：联系 Cursor support 通常无用。最稳的路径是**新邮箱 + 新付款卡 + 稳定海外 IP** 重新注册——旧账户可以继续用 Tab 模型直到订阅到期。这也是为什么 YoTrade 代刷会标配 IP 隧道。
