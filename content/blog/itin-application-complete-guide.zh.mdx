## 谁需要看这篇？

如果你是以下任何一种情况，ITIN 几乎是绕不开的第一步：

- **NRA（非美国居民）**：想申请 Chase / Citibank / Capital One 的信用卡
- **跨境创业者**：在美国注册了 LLC、要开 Mercury / Relay 商业银行账户、用 Stripe 收款
- **海外华人 / 留学生家属**：F2、H4、B1/B2 持有人需要联合报税
- **被动收入持有者**：在美国有股市、博彩、银行利息收入并产生了 1099 表

只要不属于"能直接申请 SSN"的人群（公民、绿卡、F1/J1 校内打工后申请 SSN 等），想合规享受美国金融基建，ITIN 就是入场券。

## ITIN 到底是什么？

**ITIN（Individual Taxpayer Identification Number）** 是美国国税局 IRS 发放的**税务识别号**，本质就是一个"税号"，专门给"没有资格申请 SSN，但又有报税义务"的人用。

> 💡 **关键认知**：ITIN 的设计初衷只是**报税**，不是用来申卡的。但因为部分银行愿意接受 ITIN 替代 SSN，它就顺势变成了 NRA 在美国金融系统的"准入证"。

### 有 ITIN 能干什么？

**接受 ITIN 的银行（6 家主流）**：

| 银行 | 主要价值 |
|---|---|
| American Express | 不需要 ITIN 也能申，有 ITIN 风控更松 |
| Bank of America | 同上，门槛最低 |
| Chase | ⭐ NRA 必申，5/24 规则起算锚点 |
| Citibank | ⭐ 转点 / ThankYou Points 体系入口 |
| Capital One | ⭐ Venture X / Savor 高奖励卡片 |
| Apple Card（Goldman Sachs） | iPhone 用户便捷选项 |

**不接受 ITIN 的银行**：Barclays、Discover、US Bank、Wells Fargo —— 这四家只认 SSN。

⚠️ **注意**：没 ITIN 时，你只能申 AmEx 和 BoA 两家；有了 ITIN，可申家数翻三倍，而且这三家（Chase / Citi / Capital One）恰恰是开卡奖励最丰厚、生态最完整的家。

## 申请 ITIN 的三种路径

IRS 官方允许三种申请方式，**各有各的坑**。这是本文的核心：

### 路径 A：DIY 邮寄到 IRS（最便宜，风险最大）

直接把 W-7 表格 + 税表 + **护照原件**寄到德州 IRS ITIN 处理中心。

| 维度 | 详情 |
|---|---|
| 费用 | $0（不算邮寄费） |
| 耗时 | 正常 6–8 周；报税旺季（1–4 月）可达 **12 周** |
| 护照 | **必须邮寄原件** |
| 申请人位置 | 不需要在美国 |
| 失败补救 | 拒签信寄回，重新提交 |

⚠️ **DIY 的致命风险**：护照原件邮寄过程中**丢件、毁损、IRS 内部遗失**都发生过。一旦护照丢失，IRS **不承担任何责任**，你要自行重新办护照、再次邮寄、整个流程归零，最长可能拖到半年以上。

📌 早期 IRS 接受"护照的银行公证件"，但因伪造泛滥，现在政策已收紧——**必须是护照原件或护照发放单位公证件**。中国驻外使领馆基本不提供这种公证，所以国内 NRA 走 DIY = 必须寄原件。

### 路径 B：IRS Local Office（TAC）当面办

预约 IRS 在美国本土的 Taxpayer Assistance Center，当场提交资料。

| 维度 | 详情 |
|---|---|
| 费用 | $0 |
| 耗时 | 同邮寄（材料还是要走内部流转） |
| 护照 | **不需邮寄**，工作人员现场复印归还 |
| 申请人位置 | **必须本人在美国**，需要 IRS 联合报税人也同时到场 |
| 预约要求 | 电话预约，**并非所有 TAC 都接收 ITIN 申请** |

✅ **TAC 的最大价值**：消除护照丢失风险。如果你刚好要去美国旅行 / 出差，是性价比最高的方案。

❌ **TAC 的现实限制**：要本人入境、要事先打电话确认 TAC 是否接 ITIN 业务、要把联合报税人（如果有）一起拉过去。

### 路径 C：CAA 授权代理（付费，但护照不出手）

**CAA（Certifying Acceptance Agent）** 是 IRS 授权的第三方代理。CAA 可以**核验你的护照原件**后开具核验证明，你只需要把核验证明 + W-7 + 税表寄给 IRS，**护照原件留在自己手上**。

| 维度 | 详情 |
|---|---|
| 费用 | $100–$500 不等，看代理收费 |
| 耗时 | 同邮寄路径，CAA 不加速 IRS 内部流程 |
| 护照 | ⭐ **可以不离手**（必须找有"远程核验"资质的 CAA） |
| 申请人位置 | 不需要在美国，部分 CAA 支持视频核验 |
| 失败补救 | CAA 协助补件、重新提交 |

💡 **CAA 内部分两档**：
1. **只能当面核验**的 CAA → 你还是要飞美国跟代理见面，意义不大
2. **远程视频核验 + 邮寄原件给代理 + 代理寄回护照**的 CAA → 真正帮你省心，但通常单次收 $100–$300

⚠️ **CAA 选择避坑**：申请前**务必电话或邮件确认**两件事：
- 是否需要邮寄护照原件给代理？
- 完整收费明细（部分代理会拆成"咨询费 + 核验费 + 邮寄费"分别收）

## 三种路径决策矩阵

```
你在美国 + 不怕跑一趟  →  路径 B（TAC 当面，免费）
你不在美国 + 不在乎护照风险 + 想省钱  →  路径 A（DIY 邮寄）
你不在美国 + 护照很重要 + 愿意付钱  →  路径 C（CAA 代办）⭐ 多数 NRA 的最优选
```

📌 **YoTrade 实战观察**：90% 的国内创业者最终选 CAA 路径。原因不是钱，是**护照在签证 / 出行 / 港卡 / 各种 KYC 里被反复使用**，邮寄 6–12 周等于半年不能用，机会成本远高于代办费。

## 申请 ITIN 的 8 种合法理由（W-7 表格 a–h）

填 W-7 时必须勾选申请理由。理由选错 = 直接被拒。**这是最高频的拒签原因**。

| 代码 | 适用情形 | 备注 |
|---|---|---|
| **a** | NRA 有美国来源收入要报税 | 多数情况可直接申请 SSN |
| **b** | NRA 享受税收协定优惠 | 中美 / 中港有协定，少数场景 |
| **c** | RA 有美国来源收入 | 同 a，少用 |
| **d** | **RA 的 dependent**（被抚养人） | B1/B2 父母可被在美 RA 子女 claim |
| **e** | **RA 的配偶** | F2、H4 联合报税常走这条 |
| **f** | F1 / J1 无工作但需报税 | 极少 |
| **g** | 加 / 墨 / 韩 三国护照配偶或 dependent | 仅限三国 |
| **h** | **被动收入**（NRA 走得最多）| 博彩 / 美股利息 / 1099 → ⭐ |

💡 **NRA 创业者的实战路径**：
- 如果你已注册了 **美国 LLC** + 用 Mercury / Wise 收款 → 用 **a 或 h**
- 如果你只是个人有 Robinhood / Fidelity 美股账户 → 用 **h**
- 如果你是 F2 / H4 跟着配偶在美国 → 用 **e**

## DIY 邮寄申请的具体材料

无论选哪条路径，核心三件套是固定的：

1. **W-7 表格**（IRS 官网下载，2 页）
2. **当年税表**（一般是 Form 1040；**1040NR 一般不行**，除非你是真正的 NRA 报税）
3. **护照**（原件 或 护照发放单位公证件）

📌 **细节坑**：
- W-7 要附**当年税表**，不是空白邮寄的——这是绝大多数 DIY 新手第一次被退件的原因
- 没有真实税务理由的，不要试图申请，IRS 会查
- 如果你完全没有报税理由，应该走"被动收入触发"的路径——先在美国券商开户产生 1099，再申请 ITIN

## ITIN 拿到手之后

⚠️ **必读维持规则**：拿到 ITIN 后，**每 5 年必须用此 ITIN 报税一次**，否则 IRS 会把你的 ITIN 直接吊销。

吊销后果：
- 接受 ITIN 的银行系统里你的卡可能被关
- 重新申请要走完整流程一遍

✅ **常规维持成本**：每年走一遍 Form 1040 / 5472（如果你有 LLC）就行，跟正常 LLC 报税合规一致。

## 关于 RA / NRA 身份的提醒

申请 ITIN 前，确认自己到底是 NRA 还是 RA，因为**理由代码完全不同**：

**自动 RA**：拥有美国绿卡（即使一年只在美 1 天）。

**Substantial Presence Test 计算成 RA**：
```
当年在美天数  +  去年天数 / 3  +  前年天数 / 6  ≥ 183
且当年至少 31 天
```

**F1 / J1 例外**：
- F1：在美**前 5 个完整日历年**强制 NRA
- J1：在美**前 2 个完整日历年**强制 NRA

如果你不确定自己是哪种身份，强烈建议在申请前咨询专业税务师。
